Of Taming and Breeding

This wiki provides an overview of the creatures, eggs, and offsprings introduced by the Of Taming and Breeding (OTAB) mod. It is mainly intended for players who want to understand taming mechanics, breeding behavior, and creature progression.

Note for server administrators
The information shown in this wiki is based on the default OTAB data packs. If YAML data files are modified on a server, the values shown here will not update automatically and may need to be adjusted manually.
