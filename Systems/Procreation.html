Custom Procreation System

OTAB replaces Valheim's simplified breeding behavior with a more structured procreation system that focuses on stability, believable behavior and predictable results.

Instead of constantly switching between nearby partners, creatures follow a defined reproduction cycle with a temporarily fixed partner type and stable offspring selection.

Overview: One Procreation Cycle

In OTAB, reproduction happens in clearly defined cycles.

A typical cycle consists of three stages:

  • Partner-type selection and temporary binding
  • Offspring selection
  • Pregnancy and birth

Once a cycle begins, the chosen partner type and offspring remain fixed until the cycle finishes. This prevents unpredictable changes while reproduction is already in progress.

Partner Type Binding

When reproduction begins, a creature first determines which kind of partner it will reproduce with during the current cycle.

This choice remains fixed until the cycle ends.

The purpose of this system is to prevent creatures from constantly switching between different valid partner types, which could otherwise cause unstable or confusing reproduction behavior.

This does not bind the creature to one specific individual. It only fixes the partner type for the current reproduction cycle.

Mourning and Partner Loss

If the currently selected partner type is no longer available, reproduction does not immediately switch to another valid partner type.

Instead, OTAB introduces a short waiting period before the current reproduction setup is forgotten and a new one can be selected.

This "mourning phase" prevents rapid switching between different partner types in crowded environments and keeps reproduction cycles stable.

Offspring Selection

After a partner type is chosen, the offspring type for the current cycle is selected.

Some creatures may produce different offspring depending on which partner type was selected for the current cycle.

Once the offspring type has been chosen, it remains fixed for the entire reproduction cycle.

Multiple Offspring

OTAB also allows multiple births from a single pregnancy.

After each birth, the system may continue the pregnancy and produce additional offspring before the cycle ends.

This makes it possible for creatures to produce litters rather than always giving birth to only a single offspring.

Self-Breeding Creatures

Some creatures in OTAB are able to reproduce without a partner.

In these cases the system behaves similarly to normal reproduction, but the system effectively treats the creature as its own valid partner type for the duration of the cycle.

This ensures that offspring selection and pregnancy behavior remain stable even when no partner is required.

Why OTAB Uses This System

Vanilla Valheim reproduction is simple, stable and efficient.

Most creatures in OTAB still use reproduction rules that behave very close to vanilla Valheim.

OTAB introduces fixed partner-type selection and explicit reproduction cycles mainly for creatures that require more complex breeding behavior.

This is useful when reproduction may depend on different partner types, offspring pools or special progression paths.

The system prioritizes:

  • predictable offspring results
  • stable breeding behavior
  • clear reproduction cycles
  • consistent results in more complex setups

The goal is not to increase breeding speed, but to support more advanced breeding behavior while keeping it understandable.
